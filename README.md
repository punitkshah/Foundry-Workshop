# Microsoft Foundry Aviation Workshop

Customer-ready hands-on workshop content for **Contoso Air**, focused on the Microsoft Foundry agent lifecycle:

**Build -> Evaluate -> Deploy -> Consume**

After cloning the repository, start here (paths are relative to the repository root):

- `workshop/README.md`
- `workshop/setup/prerequisites.md`
- `workshop/setup/environment-setup.md`

The repository contains:

- A 2-3 hour aviation-themed workshop
- Lab 1 for a prompt-based Foundry agent
- Lab 2 for a Microsoft Agent Framework multi-agent solution
- Instructor assets and slide content

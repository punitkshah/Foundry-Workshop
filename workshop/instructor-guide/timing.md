# Instructor Timing Guide

## Suggested Agenda (150 Minutes)

### 0:00-0:15 - Introduction

- Introduce Contoso Air scenario
- Explain Build -> Evaluate -> Deploy -> Consume
- Contrast prompt agents and multi-agent systems

### 0:15-1:10 - Lab 1

- 10 min: Foundry prompt agent setup
- 10 min: Prompt testing
- 15 min: Evaluation and prompt refinement
- 10 min: Deployment walkthrough
- 10 min: Python consumption demo

### 1:10-2:20 - Lab 2

- 10 min: Review the multi-agent architecture
- 20 min: Build specialist agents
- 20 min: Build orchestrator and run locally
- 10 min: Evaluation discussion
- 10 min: Hosted deployment walkthrough
- 10 min: Python consumption demo

### 2:20-2:35 - Wrap-up and Q&A

- Revisit the architecture evolution
- Discuss production considerations
- Answer participant questions

## Instructor Tips

- Keep Lab 1 crisp; it establishes confidence quickly.
- Spend more time on orchestration decisions in Lab 2.
- Use the evaluation discussion to connect architecture with measurable quality.

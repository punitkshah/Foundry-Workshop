# Lab 1 – Flight Delay Communications Assistant

Build a **prompt-based Microsoft Foundry agent** for DP World that turns operational delay details
into passenger-friendly communications, then evaluate, deploy, and consume it.

> **Lifecycle focus:** **Build → Evaluate → Deploy → Consume**

## Estimated Duration

45–60 minutes

## Lab Objectives

By the end of this lab, you will be able to:

- Create a Foundry project and prompt-based agent
- Define system instructions and test prompts
- Run evaluations and interpret the results
- Improve a prompt based on evaluation feedback
- Deploy the agent and consume it from Python

## Prerequisites

- Completed the [Prerequisites](../../docs/prerequisites.md) checklist
- Completed the [Environment Setup](../../docs/environment-setup.md)
- Active Azure sign-in with access to a Foundry project and model deployment
